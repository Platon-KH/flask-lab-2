# Лабораторная работа №2
## Веб-сервис "Лотерея" (вариант 20)

### Описание
RESTful веб-сервис для управления лотерейными билетами с автоматической документацией Swagger.

### Функциональность
- CRUD операции с билетами
- Сортировка по полям
- Статистика (среднее, мин, макс)
- Документация Swagger

### Запуск
```bash
cd ~
git clone https://github.com/Platon-KH/lab2-lottery.git
cd lab2-lottery
pip install -r requirements.txt
python main.py
